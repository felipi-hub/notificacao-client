# 📦 Sistema de Notificação de Clientes

Sistema web para gerenciamento de pedidos e envio de notificações via WhatsApp.

- ✅ Armazenamento local com IndexedDB
- ✅ Funciona offline
- ✅ Interface responsiva com TailwindCSS

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- TailwindCSS
- JavaScript
- IndexedDB
- Git & GitHub
